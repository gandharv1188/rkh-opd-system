# DIS-172 — Cleanup expired idempotency keys

## Summary
Scheduled cleanup job that evicts `idempotency_keys` rows older than 7 days via `DatabasePort.query`. Injectable clock and logger for testability.

## Files
- `dis/src/jobs/cleanup-idempotency.ts` — job implementation
- `dis/tests/jobs/cleanup-idempotency.test.ts` — 3 unit tests

## API
```ts
cleanupExpiredIdempotencyKeys(db, { maxAgeMs?, now?, logger? }): Promise<{ deleted, thresholdIso }>
```
- Default `maxAgeMs` = 7 days.
- Emits a single parameterised `DELETE FROM idempotency_keys WHERE created_at < $1 RETURNING 1`.
- Optional `logger.info({ deleted, thresholdIso }, 'idempotency-cleanup')` on completion.
