# DIS-173 — Fixture generator CLI

## Summary
Synthetic clinical-extraction fixture generator. Deterministic (seeded), no external deps — pure Node stdlib (`node:fs`, `node:path`, `node:process`).

## Files
- `dis/scripts/gen-fixtures.mjs` — CLI (new)

## Usage
```
node dis/scripts/gen-fixtures.mjs --count=10 --out=./tmp-dis-fixtures
```

Flags:
- `--count=<int>` — number of fixtures (default 10, must be positive integer)
- `--out=<dir>` — output directory (default `./gen-fixtures`, auto-created)

## Fixture shape
```json
{
  "schema_version": "1",
  "kind": "lab_report",
  "patient_id": "pt-gen-NNNN",
  "collected_at": "2026-04-22T00:00:00.000Z",
  "results": [ { "name": "Hb", "value": 9.69, "unit": "g/dL" }, ... ]
}
```

Tests: Hb, Glucose, HbA1c, Creatinine, WBC with unit + range-bounded value seeded by index → deterministic across runs.
