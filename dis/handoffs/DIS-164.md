# DIS-164 — Trace sampling policy

## Summary
Pure `shouldSample(ctx, policy)` helper implementing the squad's sampling policy:
- 100% on errors (`outcome === 'error'`)
- 100% on clinical-safety-tagged flows (`isClinicalSafety === true`)
- 10% on success paths by default (`DEFAULT_POLICY.successRate = 0.1`), overridable via `SamplingPolicy`

Deterministic for tests via injected `rng` on `SamplingContext`; defaults to `Math.random` in prod.

## Integration notes
Downstream tracer should construct a `SamplingContext` at span-end and gate export on `shouldSample(ctx)`. `isClinicalSafety` should be set by dose-engine / prescription-generation / PHI-adjacent spans. The policy is a pure function — safe to call in hot paths.
