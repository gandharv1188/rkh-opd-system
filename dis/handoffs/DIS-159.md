# DIS-159 — Operator-Latency SLO

## Summary
Added `computeSlo()` utility that records p50/p95 operator verify latency and
compares against an SLO target, reporting breach/warn states.

## Files
- `dis/src/observability/slo.ts` (new) — `SloConfig`, `SloReport`, `computeSlo()`
- `dis/tests/observability/slo.test.ts` (new) — 4 vitest cases
- `dis/handoffs/DIS-159.md` (this file)

## API
```ts
computeSlo(latencies: readonly number[], config: { targetMsP95: number; warnMarginMs?: number })
  => { observationCount, p50, p95, breach, warn }
```

- `breach = p95 > targetMsP95`
- `warn  = !breach && warnMarginMs !== undefined && p95 > targetMsP95 - warnMarginMs`
- Empty input returns zeroed report, no breach.

## Notes
Pure function, no I/O, no external deps. Intended to be fed the per-operator
latency series collected over a rolling window and evaluated at report time.
