## What shipped

- `dis/ui/src/components/SessionBanner.tsx` — React component that warns N ms (default 120 000) before `tokenExpiresAt`, ticks every 1 s, exposes `onRefresh` callback. Injectable `now` for tests. Exports pure helper `shouldShowWarning(nowMs, expiresAtMs, warnMs)`.
- `dis/ui/tests/session-banner.spec.ts` — Playwright test (project's chosen runner; vitest not installed). Three pure-logic cases:
  - `"appears at T-minus-2m"` (exact name required by brief) — 1.5 min remaining, 2 min window → true.
  - `"hidden outside warning window"` — 5 min remaining → false.
  - `"hidden when already expired"` — negative remaining → false.

## Design notes

- Pure helper pattern piggy-backs on existing `flags.spec.ts` style (Playwright as Node script runner).
- Component renders `role="alert"` banner with `data-testid="session-banner"` and refresh button `data-testid="session-refresh"`. Styling inline (Material amber) to match small-surface-area convention of other components.
- `secondsLeft = Math.ceil((tokenExpiresAt - nowMs) / 1000)` — ceiling so the user never sees "0s" while still in-window.
