## Design

- `RealtimeTransport` is agnostic of backend implementation — tests inject `FakeTransport`, prod uses `SseTransport`. A future Supabase Realtime adapter is trivial to add (DIS-130-followup).
- `SseTransport` relies on `EventSource`'s built-in auto-reconnect (sufficient for POC); falls back to no-op when `EventSource` is undefined (SSR / test).
- `useRealtime` connects on mount, polls `isConnected` every 1s, disconnects on unmount. Subscribe is a pass-through.
