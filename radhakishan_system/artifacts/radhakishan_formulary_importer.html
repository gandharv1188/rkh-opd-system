Accepted JSON format reference
Field names are flexible — the importer recognises common aliases. Any unrecognised fields are stored in notes. All fields except generic_name are optional.

Minimal format

Just drug name + category. Useful for quickly seeding a list.

Standard format

Name + formulations + dose + brands. The most common import format.

Full monograph format

Complete entry with dosing bands, safety, interactions, administration.

Flat dose string format

If your data has dose as a string like "40-90 mg/kg/day", the importer will parse it automatically.
