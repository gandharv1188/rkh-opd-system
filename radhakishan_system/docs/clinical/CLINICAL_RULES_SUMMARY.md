# Clinical Rules Summary

## Radhakishan Hospital — Super Pediatric OPD Prescription System

---

## 12 Core Prescription Rules (Rulebook v2, Edition 2026)

1. **Minimum 20-point format** — every prescription must contain all 20 NABH-mandated sections
2. **CAPITALS + 3-row format** — generic name in CAPS, Royal Blue, numbered, Row 1/2/3
3. **Three dose calculation methods** — A: mg/kg | B: BSA mg/m² | C: GFR-adjusted
4. **IV fluids in BLACK** — fluid type, volume ml, rate ml/hour, additives, monitoring
5. **Three mandatory safety checks** — allergy + cross-reaction + drug interaction
6. **ICD-10 + standard prescription** — every diagnosis gets a code; standard Rx generated
7. **Bilingual counselling** — Hindi + English for every medicine instruction
8. **Doctor authentication** — full name, degree, HMC reg no, date, time, emergency contacts
9. **WHO Z-scores mandatory** — WAZ, HAZ, WHZ, HCZ at every visit
10. **Developmental screening mandatory** — TDSC/DDST-II/M-CHAT-R as appropriate
11. **Immunization advice always** — IAP 2024 due/overdue vaccines at every visit
12. **A4 format, horizontal grids, multi-page acceptable**

## 3-Row Medicine Format

```
1. GENERIC NAME IN CAPITALS (Concentration — e.g. 120 mg/5 ml)
   [English] Calculated dose + route + frequency + duration + instructions
   [Hindi]   Same instructions in simple Hindi for parents
```

### Example

```
1. PARACETAMOL SUSPENSION (120 mg / 5 ml)
   1½ teaspoon (7.5 ml) orally every 6 hours as needed for fever.
   Do not give if temperature < 38°C. Max 4 doses/day.
   डेढ़ चम्मच (7.5 ml) बुखार होने पर हर 6 घंटे में मुँह से दें।
   38°C से कम तापमान पर न दें। दिन में 4 बार से ज़्यादा न दें।
```

## Dose Rounding Rules

| Formulation | Round to           | Example                    |
| ----------- | ------------------ | -------------------------- |
| Syrup       | Nearest 0.5 ml     | 4.3 ml → 4.5 ml            |
| Drops       | Nearest 0.1 ml     | 0.83 ml → 0.8 ml           |
| Tablet      | Nearest ¼ tablet   | 0.6 tab → 0.5 tab (½)      |
| Injection   | Exact to 0.1 ml    | State ml + dilution + rate |
| Insulin     | Nearest whole unit | 7.3 units → 7 units        |
| BSA drugs   | Exact              | State BSA used             |

**MAXIMUM DOSE RULE**: Calculated dose MUST NEVER exceed published maximum. Apply maximum and flag.

## Growth Charts

| Patient                                    | Chart       | Parameters                  |
| ------------------------------------------ | ----------- | --------------------------- |
| NICU/Preterm <40wks corrected              | Fenton 2013 | Weight, Length, HC — weekly |
| Preterm post-discharge until 2yr corrected | WHO 2006    | Use CORRECTED AGE           |
| Term infants 0-5 years                     | WHO 2006    | WAZ, HAZ, WHZ, HCZ          |
| Children 5-18 years                        | IAP 2015    | Height, Weight, BMI         |

**PRETERM KEY RULE**: CORRECTED AGE = Chronological age minus weeks of prematurity. Use until 2 years chronological age for all growth and developmental assessments.

**VACCINATION EXCEPTION**: Always use CHRONOLOGICAL age for vaccinations, even in preterms.

## Developmental Red Flags (IMMEDIATE REFERRAL)

- No social smile by 3 months
- No babbling by 9 months
- No words by 16 months
- No 2-word phrases by 24 months
- ANY loss of previously acquired skills at ANY age
- No head control by 4 months
- Hand dominance before 18 months
- No walking by 18 months

## Triage Scoring

| Parameter                               | Score |
| --------------------------------------- | ----- |
| Airway compromise / stridor / apnea     | 3     |
| Severe respiratory distress / cyanosis  | 3     |
| SpO₂ < 92%                              | 3     |
| Shock (cold extremities, CRT > 3 sec)   | 3     |
| Seizure / altered sensorium             | 3     |
| Severe dehydration                      | 2     |
| High fever with toxic appearance        | 2     |
| Persistent vomiting / inability to feed | 2     |
| Severe pain                             | 1     |
| Stable follow-up                        | 0     |

**Actions:** 0-1 = Routine OPD | 2-3 = Priority | 4-6 = Urgent | ≥7 = Emergency
